Вы когда-нибудь подвергались наказанию, где вас заставляли много раз писать одно и то же?
Этот вебсайт — простой для использования инструмент для наказания путём многократного написания предложений.
Он симулирует данное традиционное наказание для учеников.
Вам нужно ввести предложение, количество раз его написания и затем напечатать его столько раз, сколько указано.

ИНСТРУКЦИИ

1: Введите предложение, которое нужно напечатать несколько раз, в первое окно и нажмите ОК.

2: Введите количество раз, сколько нужно напечатать предложение, во второе окно и нажмите ОК.

3: Напечатайте предложение установленное количество раз и нажмите клавишу Enter либо кликните на кнопку Enter на экране. После выполнения задание сайт покажет, сколько раз вы написали предложение.
